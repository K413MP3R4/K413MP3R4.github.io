## 📞 Comment Appliquer les Topics

### Avec GitHub CLI
```bash
# Installer GitHub CLI
# https://cli.github.com/

# Authentifier
gh auth login

# Ajouter topics (Note: CLI n'a pas commande directe pour topics)
# Utilisez plutôt l'interface web ou l'API:

# Exemple avec curl:
curl -X PUT https://api.github.com/repos/K413MP3R4/NextGenBlock/topics \
  -H "Authorization: token $GITHUB_TOKEN" \
  -H "Accept: application/vnd.github.mercy-preview+json" \
  -d '{"names":["firewall","windows-security","network-monitoring","python","cybersecurity"]}'
```

### Via Interface Web (Recommandé)
1. https://github.com/K413MP3R4/NextGenBlock/settings
2. Cherchez "Topics" section
3. Cliquez "+ Add topics"
4. Entrez: `firewall`, `windows-security`, `network-monitoring`, `python`, `cybersecurity`
5. Click "Add"
